# Dairy Management System

A MERN Stack Dairy Management System project.

## Features
- Add Customer
- Delete Customer
- View Customer Details
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap

## Run Project

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm start
```
